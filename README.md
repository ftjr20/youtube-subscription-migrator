# ⛩️ YouTube Subscription Migrator

A secure, open-source Chrome Extension to migrate your YouTube subscriptions from an old account to a new one in controlled batches. 

*Disponível em: English, Português, 日本語.*

---

## 🚀 Features / 機能
* **Smart Extraction:** Scrapes `@handles` directly from `youtube.com/feed/channels`.
* **Autopilot Motor:** Automatically triggers subscription clicks inside YouTube's Shadow DOM.
* **Batch Safety Limit:** Protects your new account from being flagged as spam by pausing after a user-defined limit (e.g., 50 channels per day).
* **Internationalization (i18n):** Automatically detects your browser language (Supports EN, PT-BR, JA).

## 🛠️ How to Install / インストール方法

### English
1. Download this repository as a `.ZIP` file and extract it.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top-right toggle).
4. Click **Load unpacked** (top-left button) and select the extracted folder.

### Português
1. Baixe este repositório como um arquivo `.ZIP` e extraia-o.
2. Abra o Chrome e acesse `chrome://extensions/`.
3. Ative o **Modo do desenvolvedor** (canto superior direito).
4. Clique em **Compactação expandida** e selecione a pasta extraída.

### 日本語
1. このリポジトリを `.ZIP` ファイルとしてダウンロードし、解凍します。
2. Google Chrome を開き、 `chrome://extensions/` にアクセスします。
3. **デベロッパー モード**（右上）を有効にします。
4. **パッケージ化されていない拡張機能を読み込む**（左上）をクリックし、解凍したフォルダを選択します。

---

## 📖 How to Use / 使い方
1. Click the extension icon to open the **Control Panel (Dashboard)**.
2. Open your **OLD YouTube account**, go to `youtube.com/feed/channels`, scroll to the bottom, and click **Import Channels**.
3. Log into your **NEW YouTube account**.
4. Set your daily batch limit, click **Save**, and turn the **Engine ON**.
5. The extension will automatically process the channels safely!
